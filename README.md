# 🌀 Mundo Invertido

Um projeto imersivo baseado no universo de **Stranger Things**, criando uma experiência interativa sobre o Mundo Invertido. Este projeto foi desenvolvido como trabalho didático para colocar em prática conhecimentos de HTML, CSS e JavaScript.

## 📋 Sobre o Projeto

Este website oferece uma jornada temática para quem não tem medo do desconhecido. A experiência permite explorar o Mundo Invertido, assistir ao trailer da 4ª temporada de Stranger Things, visualizar uma galeria de imagens icônicas da série e participar de um formulário de inscrição para o famoso Clube de Dungeons & Dragons de Hawkings.

## ✨ Funcionalidades

- 🌓 **Alternador de Tema**: Mude entre o tema claro (Mundo Normal) e o tema escuro (Mundo Invertido)
- 🎵 **Áudio Imersivo**: Trilha sonora que muda de acordo com o tema ativo
- 📹 **Trailer Integrado**: Assista ao trailer da 4ª temporada de Stranger Things
- 🖼️ **Galeria de Imagens**: Explore imagens icônicas da série
- 📝 **Formulário de Inscrição**: Inscreva-se no Clube de D&D com seus dados de personagem
- 📱 **Design Responsivo**: Interface adaptável para diferentes tamanhos de tela
- ♿ **Acessibilidade**: Implementação de atributos ARIA para melhor acessibilidade

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Estilos e animações responsivas
- **JavaScript**: Interatividade e alternância de temas

## 🚀 Como Usar

1. Clone o repositório
2. Abra o arquivo `index.html` em seu navegador
3. Clique no botão "Inverter Mundos" para alternar entre os temas
4. Explore as seções do site e preencha o formulário do Clube de D&D

## 🎮 Seções do Site

### O Mundo Invertido
Descrição e imagem ilustrativa do Mundo Invertido, o reino sombrio controlado por Vecna.

### Trailer Stranger Things Vol. 4
Vídeo incorporado do YouTube mostrando o trailer da 4ª temporada.

### Galeria
Galeria com imagens icônicas de personagens e momentos da série.

### Clube Dungeons & Dragons
Formulário para inscrição no famoso Clube de D&D de Hawkings, com campos para:
- Nome Completo
- E-mail
- Level
- Descrição do Personagem
